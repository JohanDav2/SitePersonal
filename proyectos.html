MÁS DE NOSOTROS

Nuestros Proyectos de Desarrollo

Descubre mis proyectos más destacados en ingeniería de sistemas, donde la innovación, la eficiencia y la calidad son clave, desde desarrollos web hasta soluciones de visualización de datos.

Aplicación Móvil De Noticias - Noticias Moon

Este proyecto desarrolla una aplicación móvil con un inicio de sesión y una pantall a principal que presenta varios noticieros, proporcionando acceso inmediato a contenido informativo.

La autenticación asegura una experiencia personalizada, mientras que la interfaz organiza los noticieros de forma clara, facilitando el acceso a las noticias.

Aplicación Web - Inicio de Sesión Sencillo

Este proyecto consiste en un sistema de inicio de sesión básico desarrollado con Apps Script y Google Sheets.

Utiliza Apps Script para gestionar la autenticación de usuarios y Google Sheets para almacenar y verificar las credenciales, proporcionando un método simple y efectivo para acceder a la aplicación.

Desarrollo de Software - Gestión de Inventario

Este proyecto es una aplicación de escritorio diseñada para gestionar inventarios, proporciona herramientas para optimizar la administración de articulos.

Permite controlar y supervisar los niveles de productos, realizar seguimiento de entradas y salidas, y optimizar la administración de stock para mejorar la eficiencia operativa.
